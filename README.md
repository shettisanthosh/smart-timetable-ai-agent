# Smart Timetable AI Agent

An AI-powered academic scheduling assistant that integrates with Google Calendar to manage events, detect conflicts, and automate scheduling.

## 🚀 Features (Week 1 Completed)

- Google Calendar API Integration
- OAuth 2.0 Authentication
- Secure token storage
- View upcoming events
- Create new calendar events
- Modular project architecture
- Streamlit-based UI

## 🛠 Tech Stack

- Python
- Streamlit
- Google Calendar API
- OAuth 2.0
- Git & GitHub
